Sublime3上Markdown插件-MarkdownEditing的安装和用法(译)

发表于 2017-02-08   |     |   阅读次数

本文基于:
系统：Mac os
软件：Sublime text3
插件：MarkdownEditing

安装

通过Package Control

  • 首先安装Package Control。
  • 在sublime中，使用快捷键 CMD+SHIFT+P 打开Package Control。
  • 在弹出的输入框中输入 install package 然后点击匹配项。这时候一个可用的包列表将会显示。
  • 在可用列表的输入框中输入 MarkdownEditing 然后点击匹配项。该插件将会被下载。
  • 重启Sublime完成安装。

特点

  • 星号和下滑线自动配对来修饰选择文本。
    1.如果你输入一个空白对(如* *)然后点击backspace，两个元素都会被删除
    2.如果你输入一个空白对然后点击space，右边的元素会被删除
  • 倒引号成对出现
  • 在项目列表的尾部，按 Enter 将会自动添加新的项目列表符号
    1.在空白的项目列表点击 Tab 将会缩进项目列表符号，并且将项目列表符号切换为另一个(循环列表为 *, -, +)
    2.在空白列表按 Shift + Tab 将不会缩进，项目列表符号切换如上(我这并没有切换)
    3.支持顺序 Tab s 或 Shift + Tab s
    4.你可以在你的设置文件中将列表项目符号的自动切换撤销或者选择使用哪一个项目符号列表
    5.如果项目列表包含一个GFM 任务, 在行尾按 Enter 将会以一个新的空白任务继续
  • 在块引用(blockquote)的结尾，按 Enter 将自动延续引用
  • 选中某些文本按 > (我是按shift + >)会将选中文本改为块引用，不必选中从第一行到最后一行的全部内容，部分选中也会生效
  • 修改左侧括号可以消除选择，将光标放在你可以插入一个链接的[] 或 () 的地方(Left bracket pairing is modified to eliminate the selection and leave the cursor at a point where you can insert a [] or () pair for a link)
  • 在项目符号中显示Markdown标题(Ctrl + Shift + R). 它们以 # 开始，所以你看一眼就知道它们属于markdown文件，由于 # 的优先级，它们将在列表的顶部
  • ~ 将使用 ~~(删除线:strikethrough)包含选中文本
  • 当你想将选中的内容当做标题的时候，你可以输入 # ，多次输入可以改变标题的等级，一旦你输入6个，它将在下一次输入时重置为0，mde.match_header_hashes 决定 # 是在两边还是在行首
  • 在以hashmarks开头的行末尾键入return将在标题中插入关闭hashmarks。 这不是Markdown要求的，只是更加美观，你可以改变在你的setting中改变mde.match_header_hashes来撤销
  • Setext-style的标题可以通过 Tab 完成. 在一行仅包含= 或 - 字符按tab将会添加或者删除足够的字符去匹配上面的长度
  • 新的文档将根据第一个标题自动命名

快捷键

  • command + option + v:创建或者粘贴剪切板上的内容作为选中文本的内联链接
  • command + option + r:创建或者粘贴剪切板上的内容作为引用链接
  • command + shift + k:创建或者粘贴剪切板上的内容作为选择文本的内联图片
  • command + option + b/i:b代表粗体，i代表斜体。无论选不选中它们都会产生作用，如果没有选中，它们只会转变光标下的词，如果选中的字体已经加粗或者斜体，使用该功能会取消选种字体的加粗或者斜体。
  • command + control + 1…6:创建1-6级标题
  • option + shift + 6:添加脚注
  • command + tab:折叠或者展开当前部分(此快捷键和mac自带切换程序快捷键冲突)
  • command + shift + tab:折叠一定层级下的所有部分
  • command + option + pageup/pagedown:去当前/下一个的同等级或者更高等级的heading
  • command + shift + pageup/pagedown:去当前/下一个heading
